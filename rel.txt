%"флаг""ширина"."точность""длинна""сперицикатор"

Необходимо реализовать парсер:
1)флаг
2)ширина
3)точность
4)длинна
5)сперицикатор
Необходимо реализовать функции:
1)int is_int(char val) - возвращает 0, если символ является числом и 1 в
противном случае
2)void in_int_plus(char *str, int len, int val) - записывает число в массив с 
длинной len справо
3)void in_int_minus(char *str, int len, int val) - записывает число в массив с
длинной len слева

Так же пункт 2 и 3 для:
char
string
unsigned int
float
long float
long int
long double
short int
